bluelib
=======

SMS coding library

Game object: An object with an active screen presence. It could be a sprite or block of sprites. 
Game object format:
(var name)           (size in bytes)
SAT_index                 1
x                         1
y                         1
hspeed                    1
vspeed                    1
state                     1
flags                     1
counter_1                 1
counter_2                 1
undefined                 4
